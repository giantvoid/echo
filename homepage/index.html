Why Echo

Built for focus, not friction.

Everything you need to think and write. Nothing you don't.

Blazing fast

Rust-powered backend. Opens instantly, searches in milliseconds, stays light on your machine.

Plain Markdown files

Your notes are just .md files on disk. Live preview, no database, no lock-in.

Daily notes

Built-in calendar and Ctrl/Cmd+D jump straight to today's daily note.

Improved fuzzy search

Case-insensitive, typo-tolerant matching across titles, paths, and content — ranked so the right note surfaces first.

Quick notes

Press Ctrl/Cmd+Q to create a timestamped note and start writing immediately.

Find in note

Ctrl/Cmd+F searches inside the open note with match highlighting as you type.

Beautiful focus mode

Ctrl/Cmd+. strips away chrome for a calm, full-screen writing canvas.

10 color themes

Dark, light, solarized, hacker variants, plus five retro terminal themes with authentic pixel and CRT fonts.

Keyboard first

Ctrl/Cmd+K opens the full shortcut reference — navigate without leaving the keyboard.

Cross-platform

Native builds for Windows, macOS, and Linux — one app, same files everywhere.

Free and open source

MIT licensed. No subscriptions, no telemetry, no strings attached.

Retro themes

Terminal nostalgia, built in.

Five CRT and DOS-inspired themes ship with authentic open fonts — from IBM 3270 green phosphor to ZX Spectrum paper white and Borland blue.
