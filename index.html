Markdown Syntax

Use these common patterns to format notes while writing.

# Heading
Create headings with one or more # characters.
**bold** and *italic*
Emphasize words or phrases.
- Item or 1. Item
Create unordered or ordered lists.
[label](https://example.com)
Add links with readable text.
`inline code`
Mark short code, commands, paths, or identifiers.
> Quote
Add a block quote.
---
Add a horizontal rule.
![alt](image.png)
Reference an image; pasted images are attached automatically.
- [ ] Task and - [x] Done
Create todo items for unchecked and completed tasks.
| A | B |
| - | - |
Create tables with a header row, separator row, and data rows.
```js
code
```
Create fenced code blocks with an optional language name.
